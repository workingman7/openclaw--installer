Auto Setup

Fill in your API keys below. The installer script will handle everything else — takes 10–20 minutes.

Step 1 — Prerequisites (Manual)
⚠ These 2 things must be done manually before running the auto-installer.

Termux installed (F-Droid)

NOT Play Store — that version is outdated

APatch root granted to Termux

APatch → SuperUser → Termux → Grant

Step 2 — Your API Keys
Free at openrouter.ai → Keys → Create Key (no credit card)
From @BotFather → /newbot → copy token
From @userinfobot → /start → copy number
Step 3 — Run Auto-Installer

This generates a personalized install script with your keys pre-filled. Copy it and run in Termux.

✓
Script generated! Open Termux and paste this command:
curl -sL ... | bash
Copy

This command downloads and runs the installer inside Termux. All your keys are embedded — runs fully automatically.

Alternative: Manual copy

If curl doesn't work, tap below to copy the full script, then create the file manually in Termux:

nano ~/install_openclaw.sh
